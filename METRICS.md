# ⭐ North Star Metric

The primary product metric is:

```txt
Qualified high-savings audits completed per week
```

A qualified audit means:

✅ User completes the audit  
✅ User receives the report  
✅ Estimated savings exceed $500/month  

---

# 🧠 Why This Metric Matters

This is more meaningful than DAU because the product is not intended for daily usage.

AI spend audits are naturally:
- monthly,
- quarterly,
- or event-driven.

The real business value comes from identifying:
- high-spend startups,
- meaningful optimization opportunities,
- and qualified Credex leads.

# ⚡ Events To Instrument

The first analytics events I would track:

| Event | Purpose |
|---|---|
| 👀 Landing page viewed | Traffic measurement |
| 📝 Audit form started | Funnel entry |
| ➕ Tool added | User engagement |
| ⚙️ Audit generated | Successful completion |
| 📬 Email captured | Lead generation |
| 📄 PDF downloaded | High intent |
| 📧 Report emailed | Sharing behavior |
| 🌍 Public report opened | Viral loop tracking |
| 📞 Consultation CTA clicked | Purchase intent |
