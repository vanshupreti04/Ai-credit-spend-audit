# 📋 FAQ Content

## 🔒 Is my data private?

Yes.

The audit works using manual input and does not require connecting billing systems or sensitive financial accounts.

Public reports never expose private lead information.

---

## 👤 Does this require a login?

No.

Users can run the audit immediately without creating an account.

Email is requested only when users want to:
- download,
- share,
- send,
- or save reports.

---

## 🧠 How accurate are the recommendations?

The audit engine uses deterministic rule-based logic for:
- savings estimation,
- risk scoring,
- duplicate detection,
- and optimization recommendations.

AI is used only for:
- executive summaries,
- explanations,
- and report narration.

---

## 🤖 Which AI tools are supported?

The current audit engine supports:

- ChatGPT
- Claude
- Cursor
- GitHub Copilot
- OpenAI API
- Anthropic API
- Gemini
- Windsurf
- Perplexity

and related AI tooling categories.

---

## 🏢 Can agencies use this?

Yes.

The platform works for:
- startups,
- agencies,
- product teams,
- engineering organizations,
- and AI-heavy workflows managing multiple subscriptions and API tools.

---

# 🧠 Messaging Strategy

The landing page positioning focuses on:

✅ AI spend visibility  
✅ Fast actionable insights  
✅ Trustworthy recommendations  
✅ Founder pain points  
✅ Simple workflow  
✅ No login friction  

The messaging intentionally avoids:
- generic AI hype,
- enterprise jargon,
- and overly technical language.

---

# 🎯 Core Conversion Goal

The landing page is optimized to drive:

```txt
Audit Starts
```

rather than:
- account creation,
- newsletter signups,
- or demo requests.

The audit itself becomes the acquisition funnel.
