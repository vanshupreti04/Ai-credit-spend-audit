# 📌 Important Notes

## 🏢 Enterprise Pricing

Enterprise AI plans often do not expose public pricing.

These plans are treated inside the audit engine as:

- 🤝 Custom negotiated contracts
- 📈 Variable enterprise agreements
- 💼 Sales-assisted pricing models
- 🔒 Non-public pricing structures

Because of this, enterprise savings recommendations are more directional than exact.

---

## ⚡ API Pricing Volatility

API-based plans behave differently from seat subscriptions.

Costs can fluctuate based on:

- 🧮 Token usage
- 🚀 Request volume
- 🧠 Model selection
- 📊 Usage spikes
- 🌍 Multi-team consumption

The audit engine treats API spend as a higher-volatility category compared to fixed SaaS subscriptions.
